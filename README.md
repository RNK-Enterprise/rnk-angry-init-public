# RNK™ Angry Init

**A premium Foundry VTT module by Asgard Innovations**

## Features

Size-based initiative system for Foundry VTT with full variant rule support:

- **Size-Based Initiative** - Creatures roll initiative die based on size (Small d4, Medium d6, Large d8, Huge+ d10)
- **Bonus Action Used** - Upsize die when bonus action already spent
- **Recovery Time** - Configurable initiative recovery between rounds
- **Knockback & Reaction Blocking** - Full Variant 3 support
- **First-Round Spread** - Initiative spread logic for first round
- **GM Toolkit** - Override and adjust initiative on the fly
